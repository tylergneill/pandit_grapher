Technical Notes

Pandit Grapher code on GitHub

Key Aspects:

  • Pandit Grapher is coded in Python using the libraries NetworkX and Flask
  • The backend is structured as an ETL pipeline (see scripts) and a REST API (Swagger doc)
  • GET requests to the base route are also supported, which allows other sites to link into Pandit Grapher using Pandit ID numbers (example)
  • Graph visuals are created using the D3 JavaScript library
