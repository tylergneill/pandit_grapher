About Pandit Grapher

Pandit Grapher provides a visual way to explore relationships between works and authors in the Pandit Project prosopography database. While Pandit Grapher is in conversation with the Pandit Project, it is not officially affiliated with it.

My name is Tyler. You can learn more about my main projects at tylerneill.info/projects. Code for this project is on GitHub.

The data used is a static snapshot of the Pandit database from 2024-12-13. Only works and persons have been considered, and other entity types have been disregarded. Below are some summary statistics about the data:

  • Total entities: 16,498 (12,701 works, 3,797 authors)
  • Total relationships: 13,685 (9,287 author → work, 4,398 work → work)
