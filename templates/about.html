About Pandit Grapher

Pandit Grapher provides a visual way to explore relationships between works and authors in the Pandit Project prosopography database. Pandit Grapher is in conversation with but not officially affiliated with Pandit Project.

My name is Tyler. You can read about my main projects at tylerneill.info/projects

The data used is a static snapshot of the database from 2024-12-13. Entity types other than works and persons have been disregarded. Below are some summary statistics about the data used:

  • 16,498 total entities: 12,701 works, 3,797 authors
  • 27,370 relationships: graph edges between those entities
